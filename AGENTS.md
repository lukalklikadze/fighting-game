# Codex Instructions

Read and follow `CLAUDE.md` at the repository root before changing this project.
When working inside a feature folder, also read that folder's `CLAUDE.md`.

This is a Godot 4 two-player online fighting game. The game must remain built
for exactly two players connected over the internet, with each player using
WASD-style local controls on their own machine while the other player is synced
as the remote opponent.
