# Minigame 2 — Beer Pour

This folder is for the beer-pour minigame.

Split screen (left = you, right = opponent), networked the same way as
typeracer. **Hold SPACE/ENTER to pour; let go to ease the flow smoothly to a
stop** (a little beer still carries over from momentum, so release early). Get
as close to the brim as possible. Whoever is closer wins; spilling (overfilling
past the brim) loses to any clean pour. WINNER / LOSER (or DRAW) is shown on
each side.
